# Ecommerce Frontend Design

## Description
This project is a **desktop version of an eCommerce website** inspired by a Figma design.  
It includes a **home page**, **product listing page**, and **product details page** with hover animations, dropdowns, and interactive buttons.  

## Features
- Pixel-perfect desktop layout.
- Hero section with CTA button.
- Featured categories and products.
- Product listing with grid and pagination.
- Product details page with size dropdown and related products.
- Hover animations and interactive buttons.
- Search bar with focus effect.
- Responsive layout for desktop screens.

## How to Run
1. Download all files in the same folder (`index.html`, `products.html`, `product-details.html`, `style.css`, `main.js`).
2. Open `index.html` in any modern browser.
3. Navigation links to other pages are already included.
4. Images are linked via placeholder URLs; replace them with real images if desired.

## Technologies Used
- HTML5
- CSS3
